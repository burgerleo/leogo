# zsh 快速鍵

|      按鍵      |      功能      |
| :------------: | :------------: |
| Ctrl+A 或 Home | 移動游標到行首 |
| Ctrl+E 或 End  | 移動游標到行末 |
|     Ctrl+→     |  向右單字移動  |
|     Ctrl+←     |  向左單字移動  |
|     Ctrl+K     | 刪除游標後文字 |
|     Ctrl+U     |  刪除整列文字  |
|     Alt+D      |  刪除右側單字  |
|     Alt+W      |  刪除左側單字  |
|     Ctrl+L     |    清除螢幕    |
| Ctrl+X Ctrl+E  |    編輯命令    |
